# APRIL Capabilities

Current APRIL MVP capabilities:
- Accept typed text.
- Accept voice input from the Copilot key workflow.
- Transcribe speech locally with Whisper.
- Send prompts to the configured Ollama model for short replies.
- Speak replies through local Windows text-to-speech when voice is enabled.

Current APRIL MVP limitations:
- Tool use and device actions are not fully wired yet.
- Long-term memory is not implemented yet.
- Rich conversation history UX is still evolving.
